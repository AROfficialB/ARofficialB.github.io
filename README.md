# ⚡ Admin Dashboard — Reusable Firebase + Next.js

A private, secure admin dashboard for managing Firestore form submissions. Plug in any Firebase project config and it's ready to go.

## Features

- 🔐 Private login (Firebase Email/Password auth)
- 📊 Live submission table with search, sort, and pagination
- 📈 Metrics counter (total, filtered, latest)
- 📥 CSV export with Excel-compatible encoding
- 🔒 Strict Firestore Security Rules (public write, admin-only read)
- 🔁 100% reusable — change one config file per project
